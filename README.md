# 📝 Online Quiz Management System

## 📌 Project Overview
This is a web-based Online Quiz Management System developed using Flask and MySQL. 
It allows admin to create and manage quizzes and users to take quizzes online.

## 🚀 Features
- Admin login
- Add/Edit/Delete questions
- User registration and login
- Quiz attempt and scoring
- Result storage

## 🛠 Technologies Used
- Python
- Flask
- MySQL
- HTML
- CSS

## ▶️ How to Run the Project

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/
